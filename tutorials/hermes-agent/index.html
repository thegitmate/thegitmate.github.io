01

Install

Put Hermes on the machine

One line. The installer pulls every dependency (Python, Node, ripgrep, ffmpeg), clones the repo to ~/.hermes, and drops a global hermes command. No sudo.

terminal
# macOS or Linux, local box or a fresh VPS, same command
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

Then run the wizard once to confirm the basics:

terminal
hermes setup

Stuck anywhere in setup? Ask Claude. Paste the step you're on or the error you hit, and it'll walk you through it.

02

Model

Point it at a free DeepSeek-V4

Hermes will call any OpenAI-compatible endpoint. When base_url is set it talks to that endpoint directly. Two free options right now (July 2026):

Option A: OpenCode Zen

Free deepseek-v4-flash-free and big-pickle while they're in preview. Grab a key from the OpenCode Zen dashboard, then run the model step of the setup. Choose your provider, pick deepseek-v4-flash-free (or big-pickle), and paste your key when the wizard asks for it. That's all, it's just part of the flow.

terminal
hermes setup model

Option B: Nous Portal

Nous's own free tier, built in. One command runs the OAuth flow and hands you model access plus the tool gateway.

terminal
hermes setup --portal
03

Messaging

Wire it to Telegram

Hermes runs a gateway that bridges chat apps to the agent. We use Telegram here because it's the easiest to set up, but the same gateway also drives WhatsApp, iMessage, Discord, Slack, Signal and many more.

  1. In Telegram, message @BotFather → send /newbot → pick a name and a username ending in bot → copy the token it gives you.
  2. Message @userinfobot → it replies with your numeric user ID.

Feed both to the gateway wizard: pick Telegram, paste the token and your ID:

terminal
hermes gateway setup

Start it, then message your bot. It should answer within seconds:

terminal
hermes gateway

Once your bot is live, you can give it a profile picture. Here's a cool one.

04

Keep-alive

Make it survive reboots

On a VPS you want it always on. This registers the gateway as a background service (systemd on Linux, launchd on macOS) that starts at boot.

terminal
hermes gateway install

That's it. A self-hosted agent in your pocket, for free.

⚠ One honest word on privacy

These free endpoints are not privacy-first. OpenCode Zen's own docs say free models may retain your prompts to improve the model. So don't feed them personal, client, or secret data.

For building, coding, and everyday work (which is most of what you'll actually do) they're fast, capable, and cost nothing. Want privacy for the sensitive stuff? Point Hermes at a local model or a paid private endpoint: same config, just a different base_url.
